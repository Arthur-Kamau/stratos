## Constructor
The first function to be called when a class is initialized.
The keyword is `constructor`.
It does not have a name.

example : <br>

`class School{`<br>
 &emsp; `constructor(){`<br>
  &emsp; &emsp; `print("default constructor");`<br>
  &emsp; `}`<br/>
`}`<br>

### Named Constructors / Static Factory Methods
 Additional constructors can be defined using static function patterns or named factory methods.

`class School{`<br><br>
 &emsp; `constructor(){`<br>
  &emsp; &emsp; `print("default constructor");`<br>
  &emsp; `}`<br/><br/>
`}`<br>
