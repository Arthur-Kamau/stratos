J loops¶

There are two main types of loops

  1. for loop(commonly called foreach loop)

  2. loop

a for loop is used to loop within ranges example
for(x in a..z){
  print(x);
  // prints a , b ,c to z
}
or
for(x in 1..100){
  print(x);
  // prints 1 to 100
}

a loop without paramters equates to a while true loop ie loop{ print("hello")} will print hello forever.

looping through a list is eassy using loop.Using the keyword it to get the current index and the current value example var mList = [1,2,3,4]
loop mList{
  val  value =it.value       
  val  index =it.index       
  print (" at index $index the value is $value ")     ; 
}

for an example check /sample/loops project
